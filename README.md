# Melissa's Tennessee Real Estate Lead Generator

## Project Overview

This project is a cloud-hosted real estate lead generation application built using Microsoft Azure and GitHub. The application is designed for a licensed Tennessee REALTOR® to help buyers and sellers submit leads, receive seller preparation guidance, and interact with future AI-powered tools.

This project also serves as my cloud engineering portfolio project while I continue developing my Azure, networking, and system administration skills.
